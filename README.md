# AstralixiOS
Astralixi OS is a lightweight operating system designed for embedded and handheld devices, built for space enthusiasts and tech enthusiasts alike. With an emphasis on simplicity, performance, and a unique terminal-based experience, Astralixi OS offers an environment for users who want something a little different.

## Features
- Lightweight & Fast: Designed to work efficiently with limited resources.
- Terminal-User-Interface: A clean, fully TUI-based experience — no desktop, no clutter.
- Space & Tech Focused: Tailored for space enthusiasts, while remaining approachable for the technically curious.
- Open-Source Kernel: Built on a stripped-down, minimal Linux kernel - lean, transparent, and community-auditable.

## ⚠️ End of Life Notice

**Astralixi OS for PicoCalc has reached End of Life and is no longer actively maintained.**

A new version of Astralixi OS is in development (and/or has been released). This new iteration is written primarily in Python and is planned to support all ARM 32-bit and 64-bit devices. While targeting a broad range of ARM hardware, it is primarily aimed at the **ClockworkPi uConsole**. Stay tuned to the project repository and community channels for updates.
